# Friends Motors Website

هذا هو الموقع الرئيسي لمعرض Friends Motors.

## الملفات الرئيسية

- `index.html` - الصفحة الرئيسية
- `cars.html` - صفحة عرض السيارات
- `car-details.html` - صفحة تفاصيل السيارة

## البيانات

جميع بيانات السيارات موجودة في `assets/js/data.js`

## الصور

الصور الحالية من Google Images. يمكن استبدالها بصور حقيقية.
